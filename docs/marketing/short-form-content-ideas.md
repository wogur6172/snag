# Snag Short-Form Content Ideas

Short-form videos should show the product before explaining it. The first second should make the viewer understand the magic: real thing becomes collectible sticker.

## Creative Rules

- Show capture to cutout to collection fast.
- Keep most first tests around 6-12 seconds.
- Use captions that sound casual, not corporate.
- Show the actual phone UI when possible.
- End with a light beta CTA.
- Do not over-explain background removal.
- Do not make the video feel like an app tutorial unless it is intentionally a tutorial.

## Video Ideas

### 1. Cat Collection

Hook:

```text
I started collecting my cat like stickers.
```

Shot:

Cat on couch, quick capture, cutout, then several cat Snags on a board.

CTA:

```text
Snag beta is coming to iPhone.
```

### 2. Desk Collection

Hook:

```text
Everything on my desk became collectible.
```

Shot:

Keyboard, mug, tiny object, snack, notebook. Each becomes a Snag.

CTA:

```text
what should I snag next?
```

### 3. Friend Board

Hook:

```text
We made a board of random moments.
```

Shot:

Friend expression, coffee, receipt, silly sign, all dropped into one shared board.

CTA:

```text
send this to the friend who would ruin the board.
```

### 4. Tiny Museum

Hook:

```text
I made a tiny museum of things I like.
```

Shot:

Open blank collection, add 5 everyday objects, zoom out to a cute layout.

CTA:

```text
beta soon.
```

### 5. The First Snag

Hook:

```text
This took less than a minute.
```

Shot:

Screen recording from fresh install to first saved Snag.

CTA:

```text
iPhone beta link soon.
```

### 6. Shared Chaos

Hook:

```text
I gave my friends one empty board.
```

Shot:

Empty board, invite message, board filling with odd cutouts and text.

CTA:


### 7. Pet Sticker Shelf

CTA:

### 8. Pocket Things

Hook:

### 9. Snack/Snag Wordplay

```text

```

Shot:

Snack item becomes a Snag. Use the wordplay sparingly so the brand stays clear.

### 12. One Object A Day

Hook:

```text

```

Shot:

Simple daily collectible format.

CTA:

```text
follow the collection.
```

## First 7-Day Posting Test

Post 1-2 videos per day:

- Day 1: cat/pet and desk collection.
- Day 2: first Snag flow and tiny museum.
- Day 3: friend board and shared chaos.
- Day 4: pocket things and snack/Snag wordplay.
- Day 5: cutout fix and one-object-a-day.
- Day 6: repost the strongest concept with a better first second.
- Day 7: invite-code beta post if TestFlight public link is ready.

## What To Measure

- Which hook gets the longest watch time.
- Which video gets comments like "I want this" or "what app?"
- Which target appears naturally: pets, desk objects, or friends.
- Which CTA gets people to ask for the beta link.
- Whether people understand the shared board without explanation.
